# 🔱 Purohitam – Pooja Booking Platform

## 📖 Description

Purohitam is a full-stack web application designed to simplify and modernize the process of booking poojas and connecting with verified priests. It provides a seamless digital platform where users can explore different pooja services, check auspicious timings (muhurt), and manage their bookings efficiently. The system ensures secure authentication, email-based verification, and a smooth user experience, making traditional spiritual services more accessible through technology.

---

## 🚀 Features

* 🔐 Secure user authentication using JWT and HTTP-only cookies
* 📩 Email verification using OTP for account activation
* 🔁 Password reset functionality via email OTP
* 🛕 Pooja booking system with user-friendly interface
* 📋 “My Bookings” section to track user bookings
* 🧑‍💼 Admin dashboard for managing bookings
* 📍 Muhurt / Astrology section for auspicious timings
* 🌆 City guide for spiritual locations
* 🔔 Notification system (UI-based)
* 🚫 Access control (only verified users can book pooja)
